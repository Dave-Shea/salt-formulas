Using Cookiecutter
==================

This guide shows how to use cookiecutter template to create new Salt formula.


Installation
------------

Install in blank virtualenv.

.. code-block:: bash

    pip install cookiecutter
 
    cd cookiecutter


Usage
-----

.. code-block:: bash

    cookiecutter salt-formula
